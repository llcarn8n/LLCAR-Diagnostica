## 1. Проблемы текущей реализации

### Текущее состояние (knowledge.js)

```
[Заголовок + статус API]
[Поиск]
[Фильтр модели: All | L7 | L9]
[Категории: All | Systems | Parts | Manuals | DTC]
[Плоский список результатов]
```

### Выявленные проблемы

| # | Проблема | Влияние | Источник |
|---|----------|---------|----------|
| 1 | Плоская структура 5 табов не соответствует ментальной модели пользователя | Пользователь думает «что-то стучит в тормозах», а не «покажи мне вкладку Systems» | Интервью |
| 2 | Нет ситуационной осведомленности | Если у пользователя активен DTC P0171, раздел «Знания» об этом не знает | Архитектура |
| 3 | Нет режимов краткого/подробного просмотра | В стрессе (на трассе) пользователю нужен чеклист, а не полная статья | Интервью |
| 4 | Нет экстренного режима отображения | Критические предупреждения выглядят так же, как обычные статьи | Интервью |
| 5 | Нет источника данных (доверие) | Пользователь не видит: это из официального мануала или с форума? | Интервью |
| 6 | Нет оффлайн-индикации | Пользователь не понимает, какие статьи доступны без интернета | Интервью |
| 7 | Поиск — единственная точка входа | Пользователь должен знать, что искать. Нет browse-навигации по ситуациям | Архитектура |

## 2. Навигационная структура

### 2.1. Принцип: «Ситуация-первична, компонент-вторичен»

Вместо плоских табов используем **двухуровневую адаптивную навигацию**:

**Уровень 1 — Точки входа (что происходит?):**

```
+------------------------------------------------------+
|  [Поиск ........................................]     |
+------------------------------------------------------+
|                                                       |
|  [!] СРОЧНОЕ          [Wrench] ОБСЛУЖИВАНИЕ           |
|  Что-то не так         Плановое ТО                    |
|                                                       |
|  [Car] МОЯ МАШИНА     [Book] ИЗУЧИТЬ                  |
|  Системы и детали      Руководства                    |
|                                                       |
+------------------------------------------------------+
```

**Четыре точки входа:**

| Точка входа | Иконка | Описание | Для кого |
|-------------|--------|----------|----------|
| **Срочное** | Восклицательный знак в круге | DTC-коды, предупреждения, экстренные инструкции | «Что-то не так, помоги!» |
| **Обслуживание** | Гаечный ключ | Плановое ТО, замена расходников, подготовка к поездке | «Планирую, готовлюсь» |
| **Моя машина** | Силуэт авто | Системы, компоненты, каталог запчастей, 3D-модель | «Хочу разобраться в конкретной детали» |
| **Изучить** | Раскрытая книга | Все руководства, статьи, обучающие материалы | «Хочу узнать больше» |

**Уровень 2 — Контекстный фильтр (внутри каждой точки входа):**

Каждая точка входа раскрывается в свой набор подкатегорий. Не табами, а **вертикальным списком секций** (как в iOS Settings).

### 2.2. Адаптивная главная страница «Знаний»

Главная страница раздела «Знания» — **не пустая поисковая строка**. Она состоит из блоков, которые подстраиваются под контекст пользователя:

```
+------------------------------------------------------+
| < Знания                              [Online] API   |
+------------------------------------------------------+
| [Поиск документации...]                               |
+------------------------------------------------------+
|                                                       |
| === СЕЙЧАС ВАЖНО (если есть контекст) ===             |
| [Красная карточка: DTC P0171 — Топливная смесь]       |
| [Оранжевая: Приближается ТО-3 (50 000 км)]           |
|                                                       |
| === ДЛЯ ВАС (Top-3 рекомендации) ===                 |
| [Карточка 1: Зимняя эксплуатация L9]                  |
| [Карточка 2: Проверка тормозных колодок]               |
| [Карточка 3: Обновление ПО v4.2]                       |
|                                                       |
| === РАЗДЕЛЫ ===                                        |
| [!] Срочное (2 активных)                >             |
| [W] Обслуживание                        >             |
| [C] Моя машина                          >             |
| [B] Изучить (11 398 статей)             >             |
|                                                       |
| === НЕДАВНО ПРОСМОТРЕННОЕ ===                          |
| [Карточка: Замена масла L9...]                         |
| [Карточка: Калибровка ADAS...]                         |
|                                                       |
+------------------------------------------------------+
```

**Ключевые UX-решения:**

- Экстренная карточка открывается **в режиме чеклиста** (минимум текста, пошаговые действия)
- Первый вопрос всегда: **«Можно ли ехать?»** (из интервью: главная тревога пользователя)
- Вероятные причины визуализированы progress-барами (уже есть в diagnostics.js)
- Кнопка «Показать на 3D» ведет к digital-twin с подсвеченным компонентом
- Переход к полной статье — опциональный, не обязательный для действия

## 4. Дизайн карточек

### 4.1. Три режима отображения

Пользователь переключает режим глобально (иконка в заголовке экрана):

```
[Список]  [Карточки]  [Подробно]
  ≡          ⊞           ☰
```

### 4.2. Краткий режим (список) — для стресса

Компактные строки, минимум визуального шума. Один взгляд = понять суть.

```
┌──────────────────────────────────────────────┐
│ ● Замена тормозных колодок                    │
│   Тормоза · L9 · Мануал         Релев: 0.82  │
├──────────────────────────────────────────────┤
│ ● Прокачка тормозной системы                  │
│   Тормоза · L9 · Мануал         Релев: 0.71  │
├──────────────────────────────────────────────┤
│ ○ Тормозные колодки Li L9 — отзыв            │
│   Тормоза · L9 · drom.ru        Релев: 0.54  │
└──────────────────────────────────────────────┘
```

**Элементы строки:**

| Элемент | Описание |
|---------|----------|
| Маркер `●` / `○` | Заполненный = официальный источник, пустой = веб-источник |
| Заголовок | Жирный, 14px, одна строка с ellipsis |
| Метаданные | Серый текст, 11px: система, модель, источник |
| Релевантность | Справа, цвет по шкале (зеленый >0.7, желтый 0.4-0.7, серый <0.4) |

### 4.3. Карточный режим (по умолчанию) — баланс

```
┌──────────────────────────────────────────────┐
│ ┌─ цвет системы (4px левая полоса) ────────┐ │
│ │                                           │ │
```

### 5.2. Контексты, которые формируют рекомендации

| Контекст | Откуда берётся | Влияние на рекомендации |
|----------|---------------|------------------------|
| Активные DTC | dashboard / diagnostics | Экстренные статьи по этим кодам — приоритет #1 |

| Экран | Положение | Условие |
|-------|-----------|---------|

| **3D Twin (bottom sheet)** | Внутри sheet после информации о компоненте | При нажатии на компонент |
| **Diagnostics** | Под карточкой DTC | При просмотре конкретного DTC-кода |

### 5.4. Отличие от обычного поиска

| Параметр | Обычный поиск | Контекстные рекомендации |
|----------|--------------|--------------------------|
| Триггер | Пользователь вводит запрос | Автоматически по контексту |
| Ранжирование | По релевантности к запросу | По приоритету ситуации (DTC > ТО > сезон > обучение) |
| Количество | 10-50 результатов | Строго 3 (максимум 5) |
| Визуал | Одинаковые карточки | Пронумерованы, с объяснением «почему» |
| Обновляемость | По запросу | Реактивно при смене контекста |

### 5.5. Алгоритм формирования Top-3

```
Приоритеты (от высшего к низшему):

1. ЭКСТРЕННЫЕ (severity >= 7):
   - Статьи по активным DTC с высокой серьёзностью
   - Экстренные инструкции с [WARNING]
   → Всегда slot #1, красная карточка

2. АКТИВНЫЕ DTC (severity < 7):
   - Статьи по активным DTC
   - Вероятные причины
   → slot #1 или #2

3. ПЛАНОВОЕ ТО:
   - Если до ТО < 3000 км или < 1 месяц
   - Список работ предстоящего ТО
   → slot #2 или #3

4. СЕЗОННОЕ:
   - Если смена сезона (октябрь-ноябрь или март-апрель)
   - Подготовка к зиме/лету
   → slot #2 или #3

5. КОМПОНЕНТ (из 3D):
   - Если пользователь недавно нажимал на компонент
   - Статьи об этом компоненте
   → slot #3

6. ИСТОРИЯ:
   - Продолжение темы, которую пользователь читал
   - «Вы читали о тормозах — вот ещё по теме»
   → slot #3 (запасной)
```
