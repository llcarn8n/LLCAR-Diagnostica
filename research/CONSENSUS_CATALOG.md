# Li Auto L7/L9 — Консенсусный каталог проблем

> **Раунд 3**: Синтез результатов 6 агентов-исследователей + 6 кросс-валидаторов
> **Дата**: 2026-03-06
> **Метод**: Дедупликация, подсчёт независимых подтверждений, присвоение финального уровня уверенности

---

## Методика подсчёта

Каждый из 6 оригинальных агентов и 6 кросс-валидаторов считается независимым источником подтверждения. Агент получает +1, если нашёл проблему самостоятельно; валидатор получает +1, если подтвердил её через свои источники. Максимум = 12 (6 агентов + 6 валидаторов), но реалистичный максимум = 6-8 (не все агенты/валидаторы покрывали все темы).

- **CONFIRMED** (6+ подтверждений): бесспорная проблема, множество независимых источников
- **HIGH** (4-5 подтверждений): сильные доказательства из разнообразных источников
- **MEDIUM** (2-3 подтверждения): зафиксирована, но ограниченная перекрёстная валидация
- **LOW** (1 подтверждение): единичное упоминание, требует дополнительной проверки
- **HALLUCINATION**: опровергнуто кросс-валидаторами

### C1. Пневмоподвеска — отказ компрессора AMK / потеря высоты

- **Модели**: L7 (основная — AMK), L9 (реже — WABCO надёжнее)
- **Пробег**: 10,000-15,000 км
- **Сезон**: Зима (основной триггер), круглый год
- **Симптомы**: Автомобиль «садится», теряет клиренс. Ошибка пневмоподвески на приборной панели. Компрессор работает, но не может накачать подушки. Влага замерзает в клапанах зимой, ржавчина на подшипниках летом.
- **Системы**: Пневмоподвеска, компрессор AMK/WABCO, клапаны, пневмобаллоны
- **DTC коды**: Проприетарные (空气悬挂故障), стандартных OBD-II нет
- **Причина**: Конструктивный дефект — воздухозаборник компрессора AMK расположен под задним бампером без фильтрации. Влага, грязь и реагенты попадают напрямую. Регенерация не удаляет накопленную влагу. Деформация пластинчатого клапана (reed valve).
- **Решение**: Замена компрессора (39,000-100,000 руб). Установка фильтра Mann/Wabco (15,000-30,000 руб, снижает износ на 30-50%). Ежегодная осушка системы (10,000 руб). Замена AMK на WABCO (артикул X0129150021). Перенос воздухозаборника в салон.
- **Стоимость ремонта**: Компрессор 39,000-100,000 руб; пневмобаллон 51,000 руб/шт; фильтр 15,000-30,000 руб
- **Подтвердили**: Agent 1, Agent 2, Agent 3, Agent 4, Agent 5, Agent 6, V1, V2, V3, V5 (10/12)

### C2. Тормоза — «ватные», деградация при нагреве, неравномерный износ

- **Модели**: Both (L9 — больше жалоб)
- **Пробег**: С новья (качество торможения); 8,000 км (ускоренный износ внутренней колодки); 15,000 км (термическое растрескивание)
- **Сезон**: Все сезоны
- **Симптомы**: Тормозной путь 35 м с 60 миль/ч (Top Gear), каждое последующее торможение +3 м. Педаль «ватная», непредсказуемая после нагрева. ABS-насос «щёлкает» при торможении. Ранние L7 — редкие ZF суппорты, колодки стоят 40,000 руб.
- **Системы**: Тормозная система (суппорты, колодки, диски, ABS)
- **Причина**: Однопоршневый плавающий суппорт недостаточен для 2,520 кг. Рекуперация калибрована так, что механические тормоза используются неравномерно. Три варианта тормозных систем на L7 — невозможно определить по VIN.
- **Решение**: Замена колодок (полный комплект ~70,000 руб для ZF). Апгрейд тормозов (TEI Racing, AP Racing, Inspeed, Brembo). L9 Livis 2026 — электромеханические тормоза (EMB).

### C5. Потеря запаса хода зимой / деградация батареи

- **Модели**: Both (L7, L9)
- **Пробег**: Любой
- **Сезон**: Зима (критично ниже -20°C)
- **Симптомы**: Потеря 30-50% запаса хода зимой (с 280 до 150-180 км). Обогрев салона потребляет 8-9 кВт. Быстрая зарядка холодной батареи снижает ёмкость на 10-15% за сезон. При -30°C стоящий автомобиль разряжается за ~3 часа.
- **Системы**: ВВ батарея (CATL NMC), термоуправление, HVAC
- **Причина**: Физика литий-ионных батарей при низких температурах. Нагрузка на термоуправление. EREV-архитектура смягчает проблему (есть генератор).
- **Решение**: Предпусковой подогрев (15-20 мин). Незамерзайка -30°C. Держать заряд выше 20%. Не заряжать холодную батарею. L9 Livis 2026 — батарея 72.7 кВтч.
- **Подтвердили**: Agent 2, Agent 4, Agent 5, V1, V3, V4 (6/12)
